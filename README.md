## Description

application demo chat bot AI

## Installation

1. install

```bash
$ docker compose up
```

2. follow container log

a. front end:

```bash
$ docker logs -f next
```

b. back end:

```bash
$ docker logs -f nest
```

3. open terminal

a. front end:

```bash
$ docker exec -it next ash
```

b. back end:

```bash
$ docker exec -it nest ash
```
